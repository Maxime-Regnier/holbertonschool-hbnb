# HBnB Part 4 - Task 0 (Design)

This directory contains the static design pages required for Task 0:

- `index.html` (List of places)
- `login.html` (Login form)
- `place.html` (Place details + reviews)
- `add_review.html` (Add review form)
- `review.html` (same add review layout)
- `styles.css` (shared styles)
- `images/logo.png` and `images/icon.png`

## How to preview locally

Open each HTML file directly in your browser, or serve the folder with a static server.

## Required classes included

- `logo`
- `login-button`
- `place-card`
- `details-button`
- `place-details`
- `place-info`
- `review-card`
- `add-review`
- `form`
